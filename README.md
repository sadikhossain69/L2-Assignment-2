## **Step-1**: You need to clone this repo

    git clone https://github.com/sadikhossain69/L2-Assignment-2

## **Step-2**: You need to install node_modules

    npm install or yarn or pnpm install

    npm run start:dev or
    yarn start:dev or
    pnpm start:dev
